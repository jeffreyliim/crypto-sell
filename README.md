# Cryto Selling Service

This is a personal Binance crypto selling service which market sells all your crypto when time is of the essence.

## Installation

1. Ensure you have Golang installed correctly

2. Include the following `.env` file into the root project
```
BINANCE_API_KEY=
BINANCE_SECRET_KEY=
```

## Usage
Run this command to `Market Sell` all your existing crypto
```
go run main.go
```
